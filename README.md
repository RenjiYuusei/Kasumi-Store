# Kasumi-Store

Ứng dụng quản lý và tải xuống APK và Script game, được thiết kế với giao diện Material Design 3 hiện đại.

## Tính năng

*   **Kho ứng dụng**: Duyệt và tải xuống các ứng dụng (APK, APKS, XAPK).
*   **Kho Script**: Tìm kiếm và tải xuống các script game (Lua) cho Delta/Roblox.
*   **Giao diện hiện đại**: Sử dụng Jetpack Compose và Material Design 3.
*   **Quản lý thông minh**: Tự động phát hiện và cài đặt các định dạng file phức tạp.

## Cài đặt

Tải xuống file APK mới nhất từ mục Releases và cài đặt trên thiết bị Android của bạn.

## Phát triển

Dự án được xây dựng bằng:
*   Kotlin
*   Jetpack Compose
*   Material Design 3
*   OkHttp & Gson
*   Coil (tải ảnh)
